## 🧪 Getting Started Locally

### 📌 Prerequisites:
- Node.js
- MongoDB (local or Atlas cloud instance)

### 🔧 Backend Setup
```bash
cd server  
npm install  
npm start
💻 Frontend Setup
cd client  
npm install  
npm start
🔐 Environment Variables
In server/, create a .env file with the following:

MONGO_URI=your_mongo_connection_string  
JWT_SECRET=your_jwt_secret_key  
PORT=5000
